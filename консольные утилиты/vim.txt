vim - консольный текстовый радактор.

Помощь
:tab help - открыть help в новой вкладке, чтобы было на весь экран
:help usr_toc - оглавление руководства
:help diff - пример справки по команде
:help d - пример справки по букве
:help user-manual
:help abbreviation - сокращения
:helpgrep - поиск по хелпу по части искомого слова  :cnext :cprev  :clist - список всего найденого(технически вываливает найденое в список ошибок)  :cfir[st][!] [номер] Показать ошибку(найденое) с заданным [номером]
ctrl-] - вперёд по |ссылке|  ctrl-o/ctrl-O/ctrl-t/ctrl-T - переместиться назад
gf - открыть файл, на названии которого находится курсор; 
:set helplang=ru - переключение языка справки
Можно искать команды, соответствующие режиму. Пример сразу в :help.



D - удалить строку от курсора до конца.
а где? - удалить строку от начала до курсора

Перемещение
f/F/t/T<буква> - перепрыгнуть на букву на строке.
;/, - если недолёт или перелёт по прыжку на символ.
zh, zl - скроллить горизонтально.
ge - на конец прошлого слова.

ctrl-r - redo повторить отменённое

Окна
ctrl W ctrl r/R/x перемещать в одной группе(без плагинов)
\ww - помечаем окно, переходим в нужное и снова \ww (plugin vim-windowswap)

Вкладки
:tabnew filename
num gt/gT - перемещение по вкладкам. 

Автодополнение
ctrl-X Tab в режиме редактирования
ctrl-X ctrl-F - дополнение файлового пути

Лёгкий режим 
vim -y
ctrl L - префикс в лёгком режиме

Прямоугольная вставка
ctrl-v I пишем esc

Вставка в поиск/командную строку vim из vim копирования
ctrl-R "

Поиск 
http://vim.wikia.com/wiki/Search_for_lines_not_containing_pattern#Using_the_:v_command - поиск строк, не подпадающих под шаблон
http://vim.wikia.com/wiki/Delete_all_lines_containing_a_pattern - замена строк, не подпадающих под шаблон

Замена
:%s/oldtext/newtext/gc - заменить с подтверждением



Сравнение буферов
:windo diffthis
:windo diffoff

Сделать текущий файл readonly
set readonly
