xargs - формирование параметров

xargs — утилита для формирования списка аргументов и выполнения команды.
Команда xargs объединяет зафиксированный набор заданных в командной строке начальных аргументов с аргументами,
прочитанными со стандартного ввода, и выполняет указанную команду один или несколько раз. 

ФЛАГИ
-0 Использует во входном потоке символ-разделитель NULL ("\0") вместо "пробела" и "перевода строки", хорошо сочетается с опцией -print0 команды find

ПРИМЕРЫ

Демонстрация того, что xargs считывает данные со входящего потока и расставляет их в параметры подчинённой команде.
echo '-p tstD1/tstD2' | xargs mkdir

Каждое значение - параметр
$echo A B C | xargs echo
A B C

Каждое значение - параметр отдельной команды
$echo A B C | xargs -n 1 echo
A
B
C
